# ML4SCI HEPSIM Evaluation Test

This repository contains the solution for the ML4SCI GSoC evaluation task: **Jet Observables and Classification**.

## Contents
* `notebooks/eval_test.ipynb`: Contains the full data preprocessing, kinematic calculations (Jet Mass, Width, pT Dispersion), Lorentz boost to the rest frame, and Logistic Regression classification.

## Requirements
To run the notebook, install the required packages:
pip install -r requirements.txt

## Data Citation
Dataset provided via Zenodo: P. T. Komiske, E. M. Metodiev, J. Thaler, "Energy Flow Networks: Deep Sets for Particle Jets", JHEP 01 (2019) 121, arXiv:1810.05165.
